# Portfolio Website

## 🚀 Short Intro

This is my personal portfolio website, built to showcase my experience and projects in a clean but distinctive way.
It follows a modern-dark aesthetic with neon accents and 3D elements, reflecting both my technical profile and personal style.

🔗 **Live Demo:** https://jorge-yanas.com

## 🛠 Technologies Used

* JavaScript
* React (via npm)
* Tailwind CSS
* Three.js (3D animations)
* HTML
* Git & GitHub (version control)

## ✨ Features

* 80s-inspired modern UI (dark theme + neon pink accents)
* Interactive 3D rotating model on the landing section
* Smooth animations across the entire experience
* Desktop-first design (mobile access intentionally restricted)
* Modular and scalable component structure
* Fast and optimized performance
* Contact form with sound feedback (Cantina Band from Star Wars)

## 👤 What Users Can Do

* Explore my professional experience in a CV-like format
* Navigate through my personal projects and view them in detail
* Interact with a modern and dynamic interface
* Access project-specific external content when available

## 🧠 The Process — How I Built It

I developed this project through multiple iterations, focusing on both design and structure.

* Started with layout and user flow design
* Built the frontend entirely from scratch using React (first time using it)
* Integrated 3D elements using Three.js for a more unique experience
* Focused on performance and smooth animations
* Deployed the application on AWS with HTTPS enabled

## 📚 What I Learned

* Learned React from scratch and applied it in a real project
* Gained hands-on experience building a full frontend application
* Improved my understanding of component-based architecture
* Learned how to integrate 3D elements into a web interface
* Understood deployment processes and HTTPS configuration

## 🔧 What Could Be Improved

* Add multi-language support for better accessibility
* Improve performance of 3D elements for low-resource devices
* Design and optimize a mobile version of the site
